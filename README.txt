Este proyecto es una especie de parodia al local de comida de un amigo, para que en un futuro pueda hacerle la pagina pero de manera mas trabajada. 
Como no tengo muchos conocimientos de base de datos, se me complico bastante todo lo relacionado a este, por otro lado, estoy muy corto de tiempo por el tema de los finales y parciales de la facultad, 
por consecuencia no pude hacer la pagina mas estetica. Pero toda la pagina funciona y es acorde a lo pedido para la entrega.

Indicaciones:

Hay 3 modelos:

Clientes
Empleados
Comida

Cada uno tiene su formulario para a単adirle datos pero el unico que tiene un formulario para buscar datos es el de "Comida".

En el inicio hay dos secciones:

1- Menu: en este hay un boton para comprar una determinada comida. Al pulsarlo saldra un formulario para a単adir clientes a la base de datos y consecuentemente 
una pagina que agradecera al usuario de comprar.

2- Pasantias: Mas abajo o en el nav de la pagina aparecera un boton para "Inscribirse a una pasantia". Al pulsarlo saldra un formulario para a単adir empleados 
a la base de datos y consecuentemente una pagina que agradecera al usuario por inscribirse.

3- Para a単adir comidas se debe utilizar una url en especial que es 'adminregistercomidas/' esto es asi porque no creo que en una pagina de un negocio se deje modificar
la base de datos a los clientes.

4- En el nav hay un link llamado "Buscador de Comidas" en este se filtra la base de datos de la clase "Comida" y consecuente mente te muestra una tabla con el resultado.
